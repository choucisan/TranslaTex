<h2 id="title">TranslaTex</h2>

</div>

<p align="center">
One-click translation for LaTeX files and arXiv TeX source codes.  
Integrated with multiple translation engines (Machine Translation & AI LLMs), supporting multiple languages and APIs (more to come).
</p>

---

<h2 id="features">✨ features</h2>

- 📝 Support for translating .tex files and arXiv paper sources with local compilation
- 🔍 Automatically preserves original structure: section titles, formulas, references, etc.
- 🖥️ Clean and intuitive graphical interface (built with Gradio)
- 🌐 Multiple translation modes available: Machine Translation, AI LLM Translation — switch flexibl

## 🚀 Quick Start

### 1. Clone the Project

```bash
git clone https://github.com/choucisan/TranslaTex.git
cd TranslaTex
```

### 2. Install LaTeX
```bash
xelatex --version
bibtex --version
```

### 3. Launch GUI

```bash
python app.py
```
